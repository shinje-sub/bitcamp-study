# 06_2 - 변수와 키보드 입력 다루기

## 학습 목표

- 키보드로 입력한 값을 읽을 수 있다.
- 값을 보관하기 위해 변수를 사용할 수 있다.

```
번호:100, 차종:모닝

차량 번호 :11나1111, 차량 위치:102동 앞
```
